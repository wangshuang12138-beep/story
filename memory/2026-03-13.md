# 2026-03-13

## 项目开发：打卡应用 (Habit Tracker)

### 完成的功能

1. **修复 + 按钮不显示问题**
   - 问题：`v-if="users.length > 0"` 隐藏了整个用户切换区域
   - 解决：移除条件，让 + 按钮始终显示

2. **修复刷新 404 问题**
   - 问题：Vue Router history 模式在 GitHub Pages 不支持刷新
   - 解决：改为 hash 模式 (`createWebHashHistory`)
   - 新 URL 格式：`/#/habit/xxx`

3. **修复创建用户无响应**
   - 问题：Vercel 后端 API 无响应
   - 解决：改为纯前端版本，使用 localStorage 存储数据
   - 数据持久化：刷新不丢失，但清除缓存会丢失

4. **移动端适配**
   - 添加响应式布局 (max-width: 640px)
   - 习惯卡片单列显示
   - 按钮触摸优化（点击反馈）
   - 字体和间距适配小屏幕

### 技术栈
- 前端：Vue 3 + Vite + Pinia
- 部署：GitHub Pages (前端) + Vercel (后端，已停用)
- 数据存储：浏览器 localStorage
