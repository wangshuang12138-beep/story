# xca 的音乐故事日记

连载故事仓库，存放完整的故事内容。

## 📁 文件说明

- `story.md` - 完整连载故事（你编辑这个文件）
- `chapters.json` - 章节索引（自动生成，不要手动修改）

## ✏️ 如何更新故事

### 1. 编辑 story.md

按以下格式书写：

```markdown
## Day 1

今天的故事内容...

可以写多段。

---

## Day 2

第二天的故事...

留下一个悬念：那个蓝色的信封里到底装着什么？

---

## Day 3

（待续）
```

### 规则
- 每个章节以 `## Day N` 开头（N 是天数）
- 章节之间用 `---` 分隔
- 可以留空 `（待续）` 表示还没写

### 2. 提交到 GitHub

```bash
git add story.md
git commit -m "Add day 3 story"
git push
```

### 3. 每日展示自动同步

music-daily 会自动从本仓库获取最新故事内容。
